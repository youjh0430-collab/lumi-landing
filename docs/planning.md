## 4. 섹션별 카피 & 구성

### [1] Hero
- **헤드라인:** 오늘 퇴근하면, 집은 이미 깨끗합니다
- **서브카피:** 맞벌이 부부와 1인 가구를 위한 AI 가사 로봇 구독 서비스
- **CTA:** 무료 체험 신청하기 (14일 무료 체험)
- **카운터:** 런칭까지 D-21 + 얼리버드 혜택 안내
- **비주얼:** 로봇 메인 컷 (플레이스홀더 → 이미지 교체 예정)

### [2] 공감 (Pain Point)
- **맞벌이 카드:** "퇴근하고 왔더니 또 설거지가 쌓여있다"
- **1인 가구 카드:** "청소할 시간은 없고 미루다 보니 한 달째"
- **전환 문구:** 이제 이 상황, 루미에게 넘기세요

### [3] 솔루션 소개
- Step 1: 구독 신청 → 플랜 선택
- Step 2: 로봇 설치 (30분) → 기사 방문 설치
- Step 3: 매일 자동 작동 → 앱으로 모니터링
- **강조:** 복잡한 세팅 없음, 전담 기사 설치 지원

### [4] 핵심 기능

| 기능 | 설명 | 차별점 |
|------|------|--------|
| 자율 청소 | 공간 맵핑 후 매일 자동 청소 | 스케줄 커스터마이징 가능 |
| 빨래 분류 & 정리 | 세탁물 자동 분류, 건조 후 개기 | 타 제품 불가 기능 |
| AI 학습 | 가족 생활 패턴 학습 | 2주 후 완전 맞춤화 |
| 앱 원격 제어 | 외출 중 실시간 제어 | 이상 감지 알림 포함 |

### [6] 신뢰 요소 (Social Proof)
- 실사용자 후기 3개 (맞벌이 / 1인 가구 / 시니어)
- 수치 지표: 사용자 4,200+ 가구 · 만족도 4.8/5 · 평균 하루 2.1시간 절약
- 언론 보도 로고 배열
- **비주얼:** 사용자 프로필 사진 3장 (플레이스홀더 → 교체 예정)
